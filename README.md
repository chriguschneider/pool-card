# pool-card

A custom [Home Assistant](https://www.home-assistant.io/) Lovelace card for monitoring and controlling a pool, built with React.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The bundled card is emitted to `dist/` and can be added to Home Assistant as a Lovelace resource.
